
# Laptop Website

An E-commerce website template with most of basic functionality such as product filters and authentications utilizing Typescript, TailwindCSS, React Router, Redux, and few UI libraries. 




## Pages

- Main Page
- Feature Page
- About Us Page
- Cart Page
- Favorite Page
- Authentication Page
- Products
    - Product Detail
- Profile / Settings Page
    - Account
    - Payment Detail
    - Subscription
    - Order



## Run Locally

Clone the project

```bash
  git clone https://github.com/Twiztss/Laptop-Website
```

Go to the project directory

```bash
  cd Laptop-Website
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
